Basics

Basic use of BeautyBar means creating nice bar chart by using tools provided. This chart can then be used in blogs, part of presentations or documentation. Following steps are required to make chart:

Building chart layout

In basic use one should first choose chart template type from list. When templates is active, one can finetune it from style section to meet his needs.

Setting data

Next one needs to change default datas to new ones. This can be done either by modifying spread sheet in data section, or by importing data in CSV (comma separated values) format in file section.

Exporting chart

Final part of process is exporting created chart. This can be done by clicking Export as SVG or Export as PNG in style section. Then the chart is ready to be use in all kind of presentations. One can also save the chart to be able to modify it in the future. save and load can be found from file section.
