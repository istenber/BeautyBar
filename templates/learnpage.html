Basic use

Basic use of BeautyBar means creating nice bar chart by using tools provided. This chart can then be used in blogs, part of presentations or documentation. Following steps are required to make chart:

Setting data

In basic use one should first change default datas to new ones. This can be done either by clicking chart fields under data section and writing data values and names there, or by importing data in CSV (comma separated values) format. Import file should have six rows with two values each row comma separated.

Building chart layout

Next one starts to build the chart. It is done by choosing chart template type from list. Templates have options that can are shown when template is active. One can use options to finetune template to meet his needs.

Building chart layout can be done also, before setting data.

Saving results

Final part of process is saving and using created chart. Currently it is only possible to save the preview image from page with browser tools: right click and save as. The image is in SVG format and can be included in all kind of presentations. One can also save data and layout to BeautyBar database by choosing a name and clicking Save, and load by clicking Load. This makes it possible to continue working later on.

Export images as .png files is not yet supported.

Template layouts

Once one have succesfully created one chart, he can use its layout as a template layout for other charts. This is done by using Google Chart API (http://code.google.com/apis/chart/). One should have save chart to database. Then one can create URL following chart API specification and using saved chart name as chart name parameter (cht). This produces SVG image with old style, but with new data.

For example: http://host/chart?cht=def&chd=t:10,11,12,13,14,20&chs=250x100&chl=Hello|World|a|b|helo|worl

Template layouts are heavily under construction.
