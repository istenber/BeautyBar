Styles

Once one have succesfully created one chart, he can use its layout as a template layout (aka. style) for other charts. This is done by using Google Chart API (http://code.google.com/apis/chart/). One should have save chart to database. Then one can create URL following chart API specification and using saved chart name as chart name parameter (cht). This produces SVG image with old style, but with new data.

For example Finnish goverment debt is:
http://beauty-bar.appspot.com/chart?cht=grey&chd=t:65.7,65.7,63.2,63.1,70.6,62.9&chs=300x200&chl=2005|2006|2007|2008|Q1-09|Q2-09

Styles are heavily under construction.
