Styles

Once one have succesfully created a chart and saved it, he can use its layout as a template layout, also known as style, for other charts. Styles can be used with:

Google Chart API

Use name of a saved chart as cht value for chart API. Example chart with name grey:
http://beauty-bar.appspot.com/chart?cht=grey&chd=t:65.7...&chs=300x200&chl=2005|...

Google Docs

Create Spreadsheet document and choose Insert Gadget. Then choose custom and type address http://beauty-bar.appspot.com/spreasheet-gadget.xml and write style name and click Apply and Close.

Command line tool

One can now use BeautyBar from *nix command line, with following tool: beauty-bar.sh.
